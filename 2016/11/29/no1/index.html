PHP之验证码

发表于 2016-11-29   |   分类于 日常   |     |    热度

前言

验证码是比较常用的功能，此验证码是四位随机数字加字母的验证码，并且在验证码中添加了噪点

创建验证码背景图

<?php 
session_start();
$image = imagecreatetruecolor(100,40);
$bgcolor = imagecolorallocate($image,255,255, 255);
imagefill($image, 0, 0, $bgcolor);

数字加字母验证码

$captch_code =''; 
for($i = 0; $i < 4; $i++){
$fontsize = 6;
$fontcolor = imagecolorallocate($image, rand(0,120), rand(0,120),rand(0,120));
$data='abcdefghijklmnopqrstuvwxyz3456789';
$content = substr($data, rand(0,strlen($data)-1),1);


$captch_code.=$content;

$x = ($i*100/4)+rand(5,10);
$y=rand(5,10);
imagestring($image, $fontsize, $x, $y, $content, $fontcolor);

}
$_SESSION['authcode'] = $captch_code;

添加噪点

for($j = 0; $j < 200; $j++){
$x = rand(0,100);
$y=rand(0,30);
$color = imagecolorallocate($image, rand(100,300), rand(100,300), rand(100,300));
imagesetpixel($image, $x, $y, $color);
}

 ?>
